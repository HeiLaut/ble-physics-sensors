This is a case for a Lolin32 Lite and various magnetic field sensor modules.
For all sensors, you need to print:

Bottom_Case_Lolin32Lite.stl
Top_Case_Lolin32Lite.stl
Backplate.stl

Then, you can choose from different sensors.
For each sensor, you need to print:

..._sensor.stl
..._cap.stl

you can also choose the build plate for the LIS2MDL and the MLX90393
